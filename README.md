# WaveLab

**Create. Publish. Remix. Evolve.**

A browser-based music creation studio with visual synthesis and music theory guidance.

## Features

- **Math-based Synthesis** - Oscillators, harmonics, and envelopes you can see and manipulate
- **Visual Wave Editor** - Draw waveforms directly and hear them instantly
- **Scale Lock** - Music theory guidance that helps you stay in key
- **Effects Rack** - Reverb, delay, distortion, filters, and more
- **MIDI Support** - Connect your hardware controllers
- **Looper** - Record and layer your performances

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to start making music.

## Tech Stack

- Next.js 16
- React 19
- Web Audio API
- Tailwind CSS
